# NUCOSMOS POS

前端原型專案，現階段包含：

- React + Vite 的 POS 介面原型
- POS 系統實用架構 PRD
- Spring Boot 後端骨架

## 開發

```bash
npm install
npm run dev
```

## 後端

```bash
cd backend
mvn spring-boot:run
```

## 建置

```bash
npm run build
```
