收據抽獎 / 兌獎入口

每次結帳拿到收據後，都可以掃描 QR Code 進入抽獎頁面。系統會顯示本次是否中獎；如果沒中獎，仍可獲得 1 點， 每累積 5 點自動發一張 50 元抵用券。

你也可以直接掃描收據上的 QR Code。若已開啟本頁，輸入兌獎碼後也能查詢同一張收據。

活動規則
1. 掃描收據 QR Code 參加抽獎。
2. 若未中獎，顯示「銘謝惠顧，再接再厲」，並獲得 1 點。
3. 每累積 5 點，可獲得 50 元抵用券。
4. 五星好評、社群分享或分享 LINE 好友後出示給老闆，可直接獲得 50 元抵用券。
