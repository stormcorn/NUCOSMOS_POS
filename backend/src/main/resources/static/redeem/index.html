會員登入後參加抽獎

每次結帳拿到收據後，掃描 QR Code 或輸入兌獎碼，就可以到 NUCOSMOS 官網參加抽獎。 沒中獎可獲得 1 點，累積 5 點自動發 50 元抵用券。

查詢收據

Member Login

手機簡訊登入會員

若是第一次登入，請先填寫姓名。完成登入後，兌獎頁會自動帶入會員資料。

兌獎規則
1. 先完成手機簡訊登入，再點擊兌獎。
2. 每張收據只能兌獎一次。
3. 未中獎會顯示「銘謝惠顧，再接再厲」，並獲得 1 點。
4. 累積 5 點可獲得 50 元抵用券。
5. 在店內五星好評、分享到任意社群、或分享 LINE 好友並出示給店主，也可以獲得 50 元抵用券。
