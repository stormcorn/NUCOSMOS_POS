# 管理後台與進銷存總 Roadmap

## 1. 文件目的

這份文件是 `NUCOSMOS POS` 後台系統的唯一主 roadmap，統一追蹤：

- 管理後台
- 進銷存
- 商品、訂單、採購、供應商、報表
- 帳號、權限、系統設定

每個模組都分成：

- 已完成
- 已完成但仍可補強
- 未完成
- 暫緩

## 2. 已完成總覽

### 2.1 認證與權限

- PIN 登入
- JWT 工作階段
- `/auth/me`
- 帳號偏好儲存
- 左側選單拖曳排序
- 左側選單排序同步到後端帳號設定
- 使用者管理
- 角色管理
- 權限清單管理
- 角色權限設定
- 多角色使用者登入
- 依登入角色決定 `activeRole`
- 左側選單依權限顯示
- 路由守衛依權限控制
- 主要頁面依檢視 / 編輯權限切換

### 2.2 營運總覽與報表

- 總覽儀表板與報表中心整併
- 營運摘要卡
- 收款結構圖
- 商品分類分佈圖
- 庫存結構圖
- 異動分佈圖
- 銷售摘要
- 銷售趨勢折線圖
- 庫存 KPI
- 低庫存清單
- 庫存異動彙總
- 原料耗用摘要
- 瑕疵 / 損耗摘要
- 效期批號清單

### 2.3 商品與商品分類

- 商品管理
- 商品分類管理
- 商品圖片
- 商品配方維護
- 商品使用包裝維護
- 商品成本試算
- 商品配方版本基礎

### 2.4 進銷存

- 商品庫存管理
- 商品雙庫存模型
  - 可售庫存
  - 瑕疵庫存
- 庫存異動建立
- 補貨門檻設定
- 低庫存篩選
- 最近庫存異動紀錄
- 瑕疵報廢
- 瑕疵轉回可售
- 盤點單建立與過帳

### 2.5 原料與包裝

- 原料管理 CRUD
- 包裝管理 CRUD
- 原料庫存異動
- 包裝庫存異動
- 原料 lot 管理
- 包裝 lot 管理
- FIFO 基礎模型
- 單位換算
  - `purchaseUnit`
  - `purchaseToStockRatio`

### 2.6 訂單 / 退款 / 班次 / 裝置

- 訂單列表與詳情
- 退款資訊顯示
- POS 訂單付款後自動扣商品庫存
- POS 訂單付款後依配方扣原料
- POS 訂單付款後扣包裝
- 退款回補可售庫存
- 退款回補瑕疵庫存
- 班次管理
- 裝置管理

### 2.7 採購與供應商

- 供應商管理
- 採購單建立
- 採購收貨入庫
- 採購入庫 lot 建立
- 補貨建議
- 低庫存建議清單

## 3. 已完成但仍可補強

### 3.1 權限控制深化

- 更多細部按鈕與操作套用權限
- 後端 API 由角色型限制逐步細化到 permission 型限制
- 更完整的權限矩陣自動化測試

### 3.2 圖表與報表深化

- 毛利趨勢圖
- 訂單量趨勢圖
- 快捷時間區間切換
- 圖表匯出

### 3.3 lot / FIFO 深化

- lot 出庫分攤明細
- 更完整的 FIFO 檢核與例外處理
- lot 報表匯出

### 3.4 成本與毛利深化

- 更完整 COGS 結轉
- 批次成本追蹤
- 按商品 / 分類 / 時間區間毛利分析

## 4. 未完成功能

### 4.1 門市與系統設定

- 門市管理完整後台
- 多門市設定
- 門市層級參數設定
- 稅率設定
- 服務費設定
- 發票設定
- 付款方式設定
- 全域營運參數設定

### 4.2 操作日誌 / 審計

- 操作日誌頁
- 關鍵操作追蹤
- 變更紀錄查詢
- 帳號敏感操作審計

### 4.3 價格與促銷

- 價格管理
- 促銷規則
- 折扣限制
- 審批流程

### 4.4 匯入匯出

- 商品匯入
- 庫存匯入
- 報表匯出
- 主資料匯出

### 4.5 通知中心

- 低庫存通知
- 異常營運通知
- 裝置異常通知
- 系統公告

### 4.6 進銷存深化

- lot 預警規則
- lot / FIFO 強制策略設定
- 正式 COGS 結轉規則
- 期間成本差異分析
- 採購成本變動對毛利影響分析

## 5. 暫緩功能

以下項目目前明確不列入近期施工：

- 原料 BOM 與半成品管理
- 商品製作工序與中央廚房流程

## 6. 建議下一步

若以目前系統成熟度來看，建議優先順序如下：

1. 門市管理與系統設定
2. 操作日誌 / 審計紀錄
3. lot / FIFO 深化
4. 成本結轉與毛利分析深化
5. 價格與促銷管理
6. 匯入匯出
7. 通知中心
