# 進銷存功能 Roadmap

## 1. 目標

本文件整理 `NUCOSMOS_POS` 的進銷存功能規劃與完成狀態，作為 POS 前台、Vue 管理後台與 Spring Boot 後端的共同開發基準。

## 2. 已完成的主要功能

- 商品庫存管理
- 商品分類管理
- 瑕疵庫存管理
- 原料管理
- 包裝管理
- 雙庫存模型
  - 可售庫存
  - 瑕疵庫存
- 商品配方、包裝用量與成本計算
- POS 訂單自動扣商品、原料與包裝
- 退款回補可售或瑕疵庫存
- 正式盤點單
- 供應商管理
- 採購單與收貨入庫
- 低庫存補貨建議
- 單位換算
- 配方版本管理
- FIFO 批號 lot 管理
- 效期管理與即將到期報表
- 銷貨成本、毛利與毛利率分析
- 庫存分析報表

## 3. 第一階段：POS 訂單與盤點

### 3.1 訂單自動扣庫

已完成：

- 訂單成立後自動扣商品庫存
- 依商品配方扣除原料與包裝
- 寫入對應異動紀錄與成本資料

### 3.2 退款 / 作廢 / 退貨回補

已完成：

- 退款可回補到可售庫存
- `REFUND_DEFECT` 可回補到瑕疵庫存
- 退款時同步回補成本，更新訂單淨成本與毛利

### 3.3 正式盤點單

已完成：

- 建立盤點單
- 填寫實盤數量
- 自動產生差異，過帳為 `ADJUSTMENT_IN / ADJUSTMENT_OUT`

## 4. 第二階段：採購與補貨

### 4.1 供應商管理

已完成：

- 供應商主檔新增、編輯、停用
- 聯絡資訊與備註管理

### 4.2 採購單 / 進貨單

已完成：

- 建立採購單
- 支援原料與包裝品項
- 記錄採購單位、單位成本、批號、製造日、效期
- 收貨後自動建立批號 lot 並轉成庫存單位入庫

### 4.3 低庫存提醒 / 補貨建議

已完成：

- 低庫存項目列表
- 建議補貨數量
- 建議採購金額估算

## 5. 第三階段：成本、效期與分析

### 5.1 單位換算

已完成：

- 原料與包裝皆支援採購單位與庫存單位換算
- 以 `purchaseToStockRatio` 自動換算入庫數量與成本

### 5.2 配方版本與成本

已完成：

- 商品配方版本化
- 依配方與包裝設定計算商品成本
- 訂單成立時寫入訂單明細成本與整單 COGS

### 5.3 批號 / 效期 / FIFO

已完成：

- 原料與包裝 lot 管理
- 批號、製造日、效期欄位
- 依效期與入庫順序 FIFO 扣庫
- 即將到期 lot 報表

### 5.4 庫存與毛利報表

已完成：

- 營收摘要
- 銷貨成本、退款回補成本、淨成本
- 毛利與毛利率
- 庫存 KPI
- 商品 / 原料 / 包裝低庫存
- 原料與包裝耗用分析
- 瑕疵與損耗分析
- 異動統計

## 6. 目前不列入近期範圍

以下兩項先排除，不作為近期開發範圍：

- 原料 BOM 與半成品管理
- 商品製作工序與中央廚房流程

## 7. 後續可評估功能

若之後要再往正式商用深化，可優先評估：

1. 更完整的批號追溯與效期預警規則
2. 供應商退貨流程
3. 採購成本波動與期間毛利分析
4. 更細的報表匯出與審計追蹤
